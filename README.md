Evaluación SQL – Base de Datos Sakila

Introducción

Este repositorio contiene la resolución de la evaluación de SQL realizada sobre la base de datos Sakila.

El objetivo de esta actividad es aplicar diferentes conceptos de SQL para realizar consultas que permitan analizar información relacionada con películas, actores, categorías y alquileres dentro de una base de datos relacional.

Las consultas se han desarrollado utilizando MySQL y se han probado en MySQL Workbench.
Estructura del repositorio

El repositorio incluye:
	•	 2 Script SQL: 1  con las consultas finales correspondientes a cada ejercicio y Comentarios en el código explicando brevemente qué realiza cada consulta.
	•	               2 otro con el desarrollo paso a paso de cada ejercicio, las comprobaciones que he considerado que necesitaba y como llegue al resultado final.
            readme: con la explicacion de este.


Para ejecutar las consultas de este repositorio es necesario disponer previamente de la base de datos Sakila instalada en un servidor MySQL.

Pasos para ejecutar las consultas:
	1.	Asegurarse de tener MySQL y MySQL Workbench instalados.
	2.	Descargar e instalar la base de datos Sakila, disponible en el repositorio oficial de MySQL.
	3.	Abrir el archivo .sql de este repositorio en MySQL Workbench.
	4.	Seleccionar la base de datos Sakila antes de ejecutar las consultas: ( use.salika;)
    5.	Ejecutar cada una de las consultas incluidas en el script para visualizar los resultados correspondientes a cada ejercicio.  
             Objetivo de aprendizaje

Esta evaluación  me permite practicar y reforzar los siguientes conceptos de SQL:
	•	Consultas con SELECT
	•	Uso de JOIN para relacionar tablas
	•	Funciones de agregación (COUNT, AVG)
	•	Agrupación de datos (GROUP BY)
	•	Filtrado de agregaciones (HAVING)
	•	Uso de subconsultas
	•	Aplicación de condiciones con WHERE
